INSTALLATION
===========

- Unzip the file.
- Upload the files and folders to your server.
- if you want to create a new page, copy and rename about.html
- When you create a new page, you will have to manually add the link in the sidebar.

======
USAGE
======

Open contact.php and change email line 119:
# $to      = "youremail@domain.com";
